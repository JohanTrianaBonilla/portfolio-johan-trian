# Johan Triana Portfolio

Portafolio profesional desarrollado para presentar el perfil, proyectos, habilidades y experiencia de Johan Sebastian Triana Bonilla como estudiante de Ingenieria de Sistemas y desarrollador web full stack.

El proyecto fue construido con un enfoque moderno, visualmente limpio y orientado a destacar informacion profesional de forma clara, elegante y responsive.

## Descripcion

Este sitio funciona como carta de presentacion digital e incluye:

- Seccion principal de introduccion profesional.
- Resumen de perfil y disponibilidad.
- Proyectos destacados.
- Stack tecnico y habilidades.
- Experiencia y formacion.
- Seccion de contacto con accesos directos a redes profesionales.

## Tecnologias Utilizadas

- Next.js
- React
- TypeScript
- Tailwind CSS

## Caracteristicas

- Interfaz moderna y responsive.
- Navegacion clara por secciones.
- Soporte bilingue en espanol e ingles.
- Identidad visual profesional con paleta azul corporativa.
- Icono personalizado para navegador.
- Estructura simple y mantenible para futuras mejoras.





## Estructura del Proyecto

```text
app/
  _components/         Componentes visuales principales del portafolio
  _data/               Contenido estructurado del sitio
  globals.css          Estilos globales y variables visuales
  icon.tsx             Icono dinamico de la pestana del navegador
  layout.tsx           Layout raiz de la aplicacion
  page.tsx             Pagina principal
public/
  profile.jpg          Fotografia de perfil
  Hoja_vida_johan_triana.pdf
README.md
```

## Objetivo Del Proyecto

El objetivo de este portafolio es consolidar una presencia profesional en linea que permita mostrar capacidades tecnicas, experiencia, formacion y proyectos reales de una manera solida y atractiva para reclutadores, empresas y colaboradores.

## Estado

Proyecto activo y listo para seguir evolucionando con nuevas secciones, mejoras visuales y actualizaciones de contenido.
